Welcome To The Historical Archive Search!


The Search Button!

Clicking the 'Search' button at the top right of the page will take you to a page in which you can search the historical archive database directly. To use the search page, select the metadata categories which are relevant to your search and type in the information you want. You will then be redirected to a results page.

The Upload Button!

Clicking the 'Upload' button at the top right of the page will take you to our upload page in which you can add valuable metadata to our ever growing database. On this page click the choose file button and navigate to your desired word or excel document. Our application will then scan your document for metadata. You will get a chance to confirm this is the correct metadata you wish to upload. If there are any further questions please contact: xxxxx-xxxx

Data Policy

This website has been designed to never store the actual document transcripts on a server for any longer than a brief moment of time. When the file is uploaded, its information is passed on to a program which extracts the meta-data. Despite the file being uploaded to our server, it is never actually stored; its information is only extracted.
